# 🪐 Planit 

Planit ist ein Studyplanner für Studierende, der dabei hilft, das Studium organisiert und übersichtlich zu planen.

## Hauptfunktionen des Planers sind:
  * Die Verwaltung des Stundenplans
  * Die Verwaltung von Hausaufgaben und Abgaben
  * Die Organisation von Prüfungen und wichtigen Terminen

  Ereignisse können individuell erstellt werden. 
  Dabei lassen sich beispielsweise Modul, Dozent/in, Raum und weitere Informationen festlegen.
  Zudem gibt es Erinnerungsfunktionen für Deadlines und anstehende Aufgaben.
